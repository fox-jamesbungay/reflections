When would you want to use a remote repository rather than keeping all your work
local?

    To share work.

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    To control when changes are received so that work in progress is not overwritten.

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    Forks are clones within github, to copy from another github account as a starting point.
    Clones are copies of repositories and can be used to make updates and commit back to the source.
    Branches are copies of respositories that can be changed independently and eventually merged back together or deleted.
